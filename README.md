# WebApp
### Take data from db and print in website
1. You need to install [Docker](https://www.docker.com/)
2. Clone this project on your computer
3. With terminal go to directory WebApp
4. In terminal run
```
docker-compose up
```
### Go to http://0.0.0.0:8000/applic/
